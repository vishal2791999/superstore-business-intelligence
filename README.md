
# Superstore Business Intelligence Project

## Overview
This project demonstrates a complete Business Intelligence workflow using the Superstore sales dataset.
The analysis includes SQL queries, Excel KPI dashboards, and a Power BI dashboard.

## Tools Used
- SQL (Business queries & aggregations)
- Microsoft Excel (KPIs and charts)
- Microsoft Power BI (interactive dashboard)
- GitHub (project portfolio)

## Dataset
Superstore sales dataset containing information about:
- Orders
- Customers
- Regions
- Product categories
- Sales and profit

## Project Structure
- superstore_dataset.xlsx → Dataset
- superstore_excel_dashboard.xlsx → Excel KPI dashboard
- superstore_sql_queries.sql → SQL business queries
- superstore_project_report.pdf → Project report

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- Average Order Value

## Power BI Dashboard
The dashboard includes:
- Sales by Region (Bar Chart)
- Profit by Category (Bar Chart)
- Monthly Sales Trend (Line Chart)
- KPI Cards for Sales, Profit, and Margin

## Business Insights
- West region generates strong sales performance.
- Technology category provides higher profit margins.
- Some furniture products have lower profitability due to discounts.
